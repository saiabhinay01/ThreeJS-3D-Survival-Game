# 🎮 ThreeJS 3D Survival Game

A 3D browser game prototype built using **Three.js** and **WebGL**.

The player controls a 3D cube character, moves around a 3D environment, avoids incoming enemies, and interacts with physics-based movement and collision detection.

## 🚀 Features

- 🌐 3D scene rendering using Three.js
- 🎮 Player movement controls
- 🏃 Physics-based movement
- ⬆️ Jump mechanic
- 👾 Random enemy spawning
- 💥 Collision detection system
- 🌎 Textured 3D objects
- 💡 Lighting and shadows
- 🎥 Orbit camera controls
- 📈 Increasing difficulty over time

## 🎮 Controls

| Key | Action |
|---|---|
| A | Move Left |
| D | Move Right |
| W | Move Forward |
| S | Move Backward |
| Space | Jump |

## 🛠️ Technologies Used

- JavaScript
- Three.js
- WebGL
- HTML5
- npm
